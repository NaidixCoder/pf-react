## Instalación

### 1. Clona el repositorio:

```bash
git clone https://github.com/NaidixCoder/pf-react

```

### 2. Instala React + Vite
```bash
npm create vite@latest .
```

### 3. Instala librerias y dependencias

#### Tailwind
```bash
[npm install -D tailwindcss postcss autoprefixer]
```
